# Get-WunderlistFolder

## SYNOPSIS

## SYNTAX

```
Get-WunderlistFolder [[-Id] <Double>]
```

## DESCRIPTION
Get all Folders created by the the current user.

## EXAMPLES

### Example 1
```
PS C:\> Get-WunderlistFolder
```

Get all Folders created by the the current user.
