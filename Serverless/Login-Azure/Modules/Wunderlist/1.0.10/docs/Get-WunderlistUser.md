# Get-WunderlistUser

## SYNOPSIS
Retrieves all info related to the currently signed in Wunderlist user.

## SYNTAX

```
Get-WunderlistUser [<CommonParameters>]
```

## DESCRIPTION
Retrieves all info related to the currently signed in Wunderlist user.

## EXAMPLES

### Example 1
```
PS C:\> Get-WunderlistUser
```

Shows all information related to currently signed in Wunderlist user.
